Install
-------

Using pip
^^^^^^^^^

.. code-block:: bash

   pip install django-find-similar

From release page
^^^^^^^^^^^^^^^^^

Download source code from `GitHub Releases page <https://github.com/findsimilar/django-find-similar/releases>`_

Clone from GitHub
^^^^^^^^^^^^^^^^^

.. code-block:: bash

   git clone https://github.com/findsimilar/django-find-similar.git
   make install
